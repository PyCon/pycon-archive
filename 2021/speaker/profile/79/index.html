Graham Bleaney

Graham (@GrahamBleaney) is a Security Engineer at Facebook, focused on keeping Instagram and other Python codebases secure. He has previously spoken publicly about his work at DEF CON 28 where he gave a tutorial on Pysa, an open source Python static analyzer for security.
