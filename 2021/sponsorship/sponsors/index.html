The Python Software Foundation (PSF) is a 501(c)(3) non-profit corporation that holds the intellectual property rights behind the Python programming language. We manage the open source licensing for Python version 2.1 and later and own and protect the trademarks associated with Python. We also run the PyCon US conference annually, support several projects through our fiscal sponsorship program, support other Python conferences around the world, and fund Python related development through our grants program.

For a quick overview of the PSF's activities, please read our Annual Impact Report (https://www.python.org/psf/annual-report/2020/). For regular updates, see our blog (https://pyfound.blogspot.com/).

Sourcery helps instantly refactor Python code to make it clearer, more concise, and easier to work with/collaborate on. Sourcery integrates directly into a developer's IDE (PyCharm, VS Code, Vim, & Sublime currently) to give suggestions while a developer is working or integrates into an individual or team's GitHub repos or CI.

Toolchain is a distributed build system that leverages the OSS Pants Build project as a client.

You invoke Pants to run requirement resolution, tests, codegen, linters, formatters, typechecks and so on. It breaks that work down into many small tasks, resolves them from cache where possible, and runs the rest with as much concurrency as possible. With Toolchain, instead of being limited to the resources of your single machine, you get cloud-based caching and execution shared across your team.
