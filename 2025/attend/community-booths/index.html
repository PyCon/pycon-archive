Community Booths at PyCon US

Each year, we set aside booth space for nonprofit organizations and community open source projects that serve the Python community and the broader open source ecosystem. Join us in the Expo Hall A during the open Expo Hall hours to welcome this year's wonderful Community Booth lineup!

Black Python Devs

Our goal is to become the largest community of Black Python Developers in the world and establish our community as a source for diverse leaders in the greater Python community.

The Carpentries

The Carpentries teaches foundational coding and data science skills to researchers worldwide.

CodeDay

There's a place in tech for everyone. CodeDay is a non-profit providing welcoming and diverse opportunities for under-served students to explore a future in tech and beyond.

Code & Supply

Pittsburgh's software professional community

Django Software Foundation

Django Software Foundation supports and promotes the advancement of the Django web framework

EuroPython Society

The EuroPython Society is a non-profit organization which organizes EuroPython conferences and supports the Python community in Europe.

PyData Pittsburgh

PyData Pittsburgh is a community for data scientists, machine learning practitioners, and all professionals, students, researchers, and enthusiasts working with Python and data in Pittsburgh.

PyLadies

International mentorship group with a focus on helping more women & non-binary people become active participants and leaders in the Python open-source community.

Python Asia Organization

The Python Asia Organization (PAO) is a non-profit organization that promotes and supports the Python programming community across East and Southeast Asia. PAO organizes Python Asia Conferences, fosters leadership, streamlines event organization, and connects the community with global Python organizations like the PSF and EPS.


Python en Español

Python en Español is an initiative created by people from Spanish-speaking countries to generate a central place where the community could continue to grow and strengthen. We are a community based on respect and offer a safe place where anyone can get started with Python or improve their knowledge. In our channels, you can find activities, resources, advice, conferences, meetups, news, projects, and much more related to Python.
